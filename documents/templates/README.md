# Dokumenten-Vorlagen

Dieser Ordner enthält CI-konforme Vorlagen für:
- One-Pager / Meeting-Zusammenfassungen
- Angebote
- Präsentationen

## Verwendung

Die HTML-Dateien können im Browser geöffnet und über **Strg+P** (Drucken) → **Als PDF speichern** exportiert werden.

## Neuratex CI Farben

- **Primary (Dunkel):** `#0A1931`
- **Secondary (Blau):** `#185ADB`
- **Accent (Gold):** `#FFC947`
- **Card Background:** `#112240`
- **Text:** `#cbd5e1`
- **Headlines:** `#f8fafc`
